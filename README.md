# Trency - ¿Dónde está el tren?

Aplicacion para obtener la posicion y llegada de los proximos trenes llegando a las estaciones

## Librerias
* React
* Redux
* Immutable
* Redux sagas
* Flow
* Babel
* Google Maps
* Bulma
* Animate.css

## Code style
* Standard

## Instalacion
```
yarn install
```

## Inicializacion
```
yarn start
```
